# Trek-it 

Trek-it is a simple and visually appealing trekking and travel website built using **HTML** and **CSS**. The project focuses on showcasing trekking destinations, providing booking options, and allowing users to interact through contact and login pages. It is designed as a beginner-friendly front-end project with clean UI and responsive layouts.
